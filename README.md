# DatingApp2025
DatingApp for 2025

# Instructions
## .NET
For running the C# API project you need to navigate to `./API` and execute the commands:

1. `dotnet restore`
1. `dotnet run`

## Angular
For running the C# API project you need to navigate to `./client` and execute the commands:

1. `npm install`
1. `ng serve`
